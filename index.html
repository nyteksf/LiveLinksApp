    About

    This webpage is home to a project created with the intent of providing context for learning how to use Firebase; a "JavaScript-centric" NoSQL database service. The project idea itself behind the creation of "LiveLinks" roughly came from observing a Firebase iPhone app tutorial written in Java. As a result, said original project offered almost no guidance for construction, but was still resultantly ported to JavaScript, and with parts being modified accordingly as per taste and necessity.
